What is Bootstrap
  • Bootstrap is a free and open-source CSS framework directed
    at responsive, mobile-first front-end web development.
    It contains CSS- and JavaScript-based design templates for typography,
    forms, buttons, navigation, and other interface components.
History
  • Bootstrap was developed by Mark Otto and Jacob Thornton at Twitter in August 2011.
  • Bootstrap 2.x was released on 31st January 2012, with builtin support for Glyficons.
  • Bootstrap 3 was released on 19 August, 2013.
  • V3 focussed on mobile first approach.
  • Bootstrap 4 was announced on 29th October and stable release was on 18th January 2018.
  • Stable version is 4.5

List
  • Responsive list groups built with the latest Bootstrap 5.
    List groups are responsive components for displaying a
    series of content. A beautiful alternative to unordered lists.
    List groups are a flexible and powerful component
    for displaying a series of content. Modify and extend
    them to support just about any content within.

    • Active items

    Add .active to a .list-group-item to indicate the current active selection.

    Disabled items

    Add .disabled to a .list-group-item to make it appear disabled. Note that some elements with .disabled will also require custom JavaScript to fully disable their click events (e.g., links).

Grid System
  • Bootstrap divides screen into 12 equal columns.
  • Screen sizes (width on px)
    1. es -> size < 576px
    2. sm -> size >= 576px
    3. md -> size >= 768px
    4. lg -> size >= 992px
    5. xl -> size >= 1200px
    6. xxl -> size >= 1400px
